void Flight::printInfo ( int  dMode,
int  sMode 
)

Prints out flight information. Recurses to Seat.printInfo when applicable.

Parameters
dMode: determins how much information is printed.
A mode of 0 prints only flight-specific info.
A mode of 1 prints out flight info and seat avalibility.
A mode of 2 prints out flight info and all passengers
sMode: determins how passengers are sorted when displayed.
A mode of 0 sorts them numerically.
A mode of 1 sorts them alphabetically.
